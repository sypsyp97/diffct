Filtered Backprojection - Fan Beam

This example demonstrates 2D fan beam reconstruction using the Filtered Backprojection (FBP) algorithm with diffct.

Overview

Fan beam geometry is commonly used in medical CT scanners where X-rays emanate from a point source to a linear detector array. This example shows:

  • Creating a Shepp-Logan phantom

  • Computing fan beam forward projection

  • Applying fan beam weighting and filtering

  • Performing backprojection with geometric corrections

Code

  1import math
  2import numpy as np
  3import torch
  4import matplotlib.pyplot as plt
  5from diffct.differentiable import FanProjectorFunction, FanBackprojectorFunction
  6
  7
  8def shepp_logan_2d(Nx, Ny):
  9    Nx = int(Nx)
 10    Ny = int(Ny)
 11    phantom = np.zeros((Nx, Ny), dtype=np.float32)
 12    ellipses = [
 13        (0.0, 0.0, 0.69, 0.92, 0, 1.0),
 14        (0.0, -0.0184, 0.6624, 0.8740, 0, -0.8),
 15        (0.22, 0.0, 0.11, 0.31, -18.0, -0.8),
 16        (-0.22, 0.0, 0.16, 0.41, 18.0, -0.8),
 17        (0.0, 0.35, 0.21, 0.25, 0, 0.7),
 18    ]
 19    cx = (Nx - 1)*0.5
 20    cy = (Ny - 1)*0.5
 21    for ix in range(Nx):
 22        for iy in range(Ny):
 23            xnorm = (ix - cx)/(Nx/2)
 24            ynorm = (iy - cy)/(Ny/2)
 25            val = 0.0
 26            for (x0, y0, a, b, angdeg, ampl) in ellipses:
 27                th = np.deg2rad(angdeg)
 28                xprime = (xnorm - x0)*np.cos(th) + (ynorm - y0)*np.sin(th)
 29                yprime = -(xnorm - x0)*np.sin(th) + (ynorm - y0)*np.cos(th)
 30                if xprime*xprime/(a*a) + yprime*yprime/(b*b) <= 1.0:
 31                    val += ampl
 32            phantom[ix, iy] = val
 33    phantom = np.clip(phantom, 0.0, 1.0)
 34    return phantom
 35
 36def ramp_filter(sinogram_tensor):
 37    device = sinogram_tensor.device
 38    num_views, num_det = sinogram_tensor.shape
 39    freqs = torch.fft.fftfreq(num_det, device=device)
 40    omega = 2.0 * torch.pi * freqs
 41    ramp = torch.abs(omega)
 42    ramp_2d = ramp.reshape(1, num_det)
 43    sino_fft = torch.fft.fft(sinogram_tensor, dim=1)
 44    filtered_fft = sino_fft * ramp_2d
 45    filtered = torch.real(torch.fft.ifft(filtered_fft, dim=1))
 46    
 47    return filtered
 48
 49def main():
 50    Nx, Ny = 256, 256
 51    phantom = shepp_logan_2d(Nx, Ny)
 52    num_angles = 360
 53    angles_np = np.linspace(0, 2*math.pi, num_angles, endpoint=False).astype(np.float32)
 54
 55    num_detectors = 600
 56    detector_spacing = 1.0
 57    source_distance = 800.0
 58    isocenter_distance = 500.0
 59
 60    device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
 61    image_torch = torch.tensor(phantom, device=device, requires_grad=True)
 62    angles_torch = torch.tensor(angles_np, device=device)
 63
 64    sinogram = FanProjectorFunction.apply(image_torch, angles_torch, num_detectors,
 65                                          detector_spacing, source_distance, isocenter_distance)
 66
 67    # --- FBP weighting and filtering ---
 68    # For fan-beam FBP, projections must be weighted before filtering.
 69    # Weight = cos(gamma), where gamma is the fan angle for each detector.
 70    u = (torch.arange(num_detectors, dtype=image_torch.dtype, device=device) - (num_detectors - 1) / 2) * detector_spacing
 71    gamma = torch.atan(u / source_distance)
 72    weights = torch.cos(gamma).unsqueeze(0)  # Shape (1, num_detectors) for broadcasting
 73
 74    # Apply weights before filtering
 75    sino_weighted = sinogram * weights
 76    sinogram_filt = ramp_filter(sino_weighted).detach().requires_grad_(True).contiguous()
 77
 78    reconstruction = FanBackprojectorFunction.apply(sinogram_filt, angles_torch,
 79                                                    detector_spacing, Nx, Ny,
 80                                                    source_distance, isocenter_distance)
 81    
 82    # --- FBP normalization ---
 83    # The backprojection is a sum over all angles. To approximate the integral,
 84    # we need to multiply by the angular step d_beta.
 85    # The fan-beam FBP formula also includes a factor of 1/2 when integrating over [0, 2*pi].
 86    # d_beta = 2 * pi / num_angles
 87    # Normalization factor = (1/2) * d_beta = pi / num_angles
 88    reconstruction = reconstruction * (math.pi / num_angles)
 89
 90    loss = torch.mean((reconstruction - image_torch)**2)
 91    loss.backward()
 92
 93    print("Loss:", loss.item())
 94    print("Center pixel gradient:", image_torch.grad[Nx//2, Ny//2].item())
 95
 96    sinogram_cpu = sinogram.detach().cpu().numpy()
 97    reco_cpu = reconstruction.detach().cpu().numpy()
 98
 99    plt.figure(figsize=(12,4))
100    plt.subplot(1,3,1)
101    plt.imshow(phantom, cmap='gray')
102    plt.title("Phantom")
103    plt.axis('off')
104    plt.subplot(1,3,2)
105    plt.imshow(sinogram_cpu, cmap='gray', aspect='auto')
106    plt.title("Fan Sinogram")
107    plt.axis('off')
108    plt.subplot(1,3,3)
109    plt.imshow(reco_cpu, cmap='gray')
110    plt.title("Fan Reconstruction")
111    plt.axis('off')
112    plt.tight_layout()
113    plt.show()
114
115    # print data range of the phantom and reco
116    print("Phantom range:", phantom.min(), phantom.max())
117    print("Reco range:", reco_cpu.min(), reco_cpu.max())
118
119if __name__ == "__main__":
120    main()

Key Concepts

  1. Fan Beam Geometry: Rays diverge from a point source to detector elements, creating geometric magnification.

  2. Weighting: Fan beam projections must be weighted by cos(γ) where γ is the fan angle for each detector.

  3. Geometric Parameters: - Source distance: Distance from X-ray source to isocenter - Isocenter distance: Distance from isocenter to detector

  4. Normalization: Similar to parallel beam, but accounts for the 2π angular range.

Usage

python examples/fbp_fan.py

The example will display the phantom, sinogram, and reconstructed image.
