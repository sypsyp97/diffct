2D Parallel Beam Filtered Backprojection (FBP)

This example demonstrates how to use the ParallelProjectorFunction and ParallelBackprojectorFunction from diffct to perform filtered backprojection (FBP) reconstruction in a 2D parallel-beam geometry.

Mathematical Background

Parallel Beam Geometry

In parallel beam CT, X-rays are collimated into parallel beams. The projection at angle \(\theta\) and detector position \(t\) is given by the Radon transform:

\[p(t, \theta) = \int_{-\infty}^{\infty} f(t\cos\theta - s\sin\theta, t\sin\theta + s\cos\theta) \, ds\]

where \(f(x,y)\) is the 2D attenuation coefficient distribution.

Filtered Backprojection Algorithm

The FBP reconstruction formula is:

\[f(x,y) = \int_0^\pi p_f(x\cos\theta + y\sin\theta, \theta) \, d\theta\]

where \(p_f(t, \theta)\) is the filtered projection:

\[p_f(t, \theta) = \int_{-\infty}^{\infty} p(t', \theta) h(t - t') \, dt'\]

The ramp filter \(h(t)\) in frequency domain is:

\[H(\omega) = |\omega|\]

Implementation Details

The example implements the following steps:

  1. Forward Projection: Compute the Radon transform using ParallelProjectorFunction

  2. Ramp Filtering: Apply the ramp filter \(H(\omega) = |\omega|\) in frequency domain

  3. Backprojection: Reconstruct using the filtered projections with ParallelBackprojectorFunction

  4. Normalization: Apply the factor \(\frac{\pi}{N_{\text{angles}}}\) to approximate the continuous integral

Phantom Description

The Shepp-Logan phantom consists of 5 ellipses with different attenuation coefficients, designed to simulate brain tissue structures. Each ellipse is defined by:

  • Center position \((x_0, y_0)\)

  • Semi-axes lengths \(a, b\)

  • Rotation angle \(\phi\)

  • Attenuation value \(A\)

2D Parallel Beam FBP Example
  1import numpy as np
  2import torch
  3import matplotlib.pyplot as plt
  4from diffct.differentiable import ParallelProjectorFunction, ParallelBackprojectorFunction
  5
  6
  7def shepp_logan_2d(Nx, Ny):
  8    Nx = int(Nx)
  9    Ny = int(Ny)
 10    phantom = np.zeros((Nx, Ny), dtype=np.float64)
 11    ellipses = [
 12        (0.0, 0.0, 0.69, 0.92, 0, 1.0),
 13        (0.0, -0.0184, 0.6624, 0.8740, 0, -0.8),
 14        (0.22, 0.0, 0.11, 0.31, -18.0, -0.8),
 15        (-0.22, 0.0, 0.16, 0.41, 18.0, -0.8),
 16        (0.0, 0.35, 0.21, 0.25, 0, 0.7),
 17    ]
 18    cx = (Nx - 1) / 2
 19    cy = (Ny - 1) / 2
 20    for ix in range(Nx):
 21        for iy in range(Ny):
 22            xnorm = (ix - cx) / (Nx / 2)
 23            ynorm = (iy - cy) / (Ny / 2)
 24            val = 0.0
 25            for (x0, y0, a, b, angdeg, ampl) in ellipses:
 26                th = np.deg2rad(angdeg)
 27                xprime = (xnorm - x0) * np.cos(th) + (ynorm - y0) * np.sin(th)
 28                yprime = -(xnorm - x0) * np.sin(th) + (ynorm - y0) * np.cos(th)
 29                if xprime * xprime / (a * a) + yprime * yprime / (b * b) <= 1.0:
 30                    val += ampl
 31            phantom[ix, iy] = val
 32    phantom = np.clip(phantom, 0.0, 1.0)
 33    return phantom
 34
 35def ramp_filter(sinogram_tensor):
 36    device = sinogram_tensor.device
 37    num_views, num_det = sinogram_tensor.shape
 38    freqs = torch.fft.fftfreq(num_det, device=device)
 39    omega = 2.0 * torch.pi * freqs
 40    ramp = torch.abs(omega)
 41    ramp_2d = ramp.reshape(1, num_det)
 42    sino_fft = torch.fft.fft(sinogram_tensor, dim=1)
 43    filtered_fft = sino_fft * ramp_2d
 44    filtered = torch.real(torch.fft.ifft(filtered_fft, dim=1))
 45    
 46    return filtered
 47
 48def main():
 49    Nx, Ny = 256, 256
 50    phantom = shepp_logan_2d(Nx, Ny)
 51    num_angles = 360
 52    angles_np = np.linspace(0, 2*np.pi, num_angles, endpoint=False).astype(np.float32)
 53
 54    num_detectors = 512
 55    detector_spacing = 1.0
 56
 57    device = torch.device('cuda' if torch.cuda.is_available() else 'cpu')
 58    image_torch = torch.tensor(phantom, device=device, requires_grad=True)
 59    angles_torch = torch.tensor(angles_np, device=device, requires_grad=False)
 60
 61    sinogram = ParallelProjectorFunction.apply(image_torch, angles_torch,
 62                                               num_detectors, detector_spacing)
 63    
 64    sinogram_filt = ramp_filter(sinogram).detach().requires_grad_(True).contiguous() 
 65
 66    reconstruction = ParallelBackprojectorFunction.apply(sinogram_filt, angles_torch,
 67                                                         detector_spacing, Nx, Ny)
 68    
 69    # --- FBP normalization ---
 70    # The backprojection is a sum over all angles. To approximate the integral,
 71    # we need to multiply by the angular step d_theta.
 72    # The FBP formula also includes a factor of 1/2 when integrating over [0, 2*pi].
 73    # d_theta = 2 * pi / num_angles
 74    # Normalization factor = (1/2) * d_theta = pi / num_angles
 75    reconstruction = reconstruction * (np.pi / num_angles)
 76
 77    loss = torch.mean((reconstruction - image_torch)**2)
 78    loss.backward()
 79
 80    print("Loss:", loss.item())
 81    print("Phantom gradient center pixel:", image_torch.grad[Nx//2, Ny//2].item())
 82    print("Reconstruction shape:", reconstruction.shape)
 83
 84    sinogram_cpu = sinogram.detach().cpu().numpy()
 85    reco_cpu = reconstruction.detach().cpu().numpy()
 86
 87    plt.figure(figsize=(12,4))
 88    plt.subplot(1,3,1)
 89    plt.imshow(phantom, cmap='gray')
 90    plt.title("Phantom")
 91    plt.axis('off')
 92    plt.subplot(1,3,2)
 93    plt.imshow(sinogram_cpu, aspect='auto', cmap='gray')
 94    plt.title("Differentiable Sinogram")
 95    plt.axis('off')
 96    plt.subplot(1,3,3)
 97    plt.imshow(reco_cpu, cmap='gray')
 98    plt.title("Differentiable Recon")
 99    plt.axis('off')
100    plt.tight_layout()
101    plt.show()
102
103    # print data range of the phantom and reco 
104    print("Phantom range:", phantom.min(), phantom.max())
105    print("Reco range:", reco_cpu.min(), reco_cpu.max())
106
107if __name__ == "__main__":
108    main()
